# Курсова робота: Система управління студентами

## Опис проекту
Цей проект є консольним додатком на C++ для управління даними студентів-дипломників. Програма розроблена для демонстрації принципів об'єктно-орієнтованого програмування (інкапсуляція, успадкування, поліморфізм), роботи з файловою системою, обробки виняткових ситуацій та використання поведінкових патернів проектування (Ітератор).

## Основний функціонал
- **Облік студентів**: Додавання, видалення та відображення студентів з базовою інформацією (ПІБ, курс, номер заліковки, рейтинг) та даними про дипломну роботу (тема, відсоток виконання).
- **Редагування даних**: Зручне оновлення інформації про конкретного студента за його ID.
- **Автоматичне збереження (Auto-sync)**: Усі зміни автоматично зберігаються та завантажуються з файлу `data.csv`.
- **Перевантаження операторів**: Використання унарного оператора `++` для переведення студента на наступний курс та бінарного `^` для зміни рейтингу.
- **Фільтрація даних**: Швидкий пошук та виведення списку студентів, які мають високий відсоток виконання дипломної роботи (>80%).
- **Пізнє зв'язування**: Демонстрація поліморфізму під час виконання програми за допомогою віртуальних методів.

## Використані технології
- **Мова програмування**: C++ (стандарт C++11 і вище)
- **Стандартна бібліотека C++ (STL)**: `iostream`, `string`, `vector`, `fstream`, `sstream`, `stdexcept`

## Встановлення та запуск

1. Склонуйте репозиторій на свій комп'ютер.
2. Переконайтеся, що у вас встановлено компілятор C++ (наприклад, GCC, Clang або MSVC).
3. Відкрийте термінал у папці з проектом та скомпілюйте файл `main_c++_work.c++`. Приклад для `g++`:
   ```bash
   g++ main_c++_work.c++ -o student_manager
   ```
4. Запустіть скомпільовану програму:
   - На Windows:
     ```cmd
     student_manager.exe
     ```
   - На Linux / macOS:
     ```bash
     ./student_manager
     ```

## Структура проекту
- `main_c++_work.c++` — головний файл з вихідним кодом програми.
- `data.csv` — локальна база даних (створюється автоматично при першому збереженні студента).
